## About

Cassette tape loop simulation, built using JUCE framework. Has real-time adjustable playback speed. Replays audio recorded from the hostâ€™s audio input device. Has unit tests to verify tape contents after processing through various methods.

## How to build & run

 - Download and install the JUCE framework
 - Checkout this repository, then open 'tapeplayervst.jucer' in Projucer
 - Press 'Save and Open in IDE' within Projucer to open the project in your IDE of choice, then run a build from there
